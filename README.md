Está separado de acordo com a data da descoberta e será sinalizado da seguinte forma:

- :coffee: - back-end
- :beers: - front-end
- :pizza: - design
- :computer: - arquitetura e ferramentas
- :iphone: - mobile
- :bulb: - ideias
- :zap: - outros/etc
